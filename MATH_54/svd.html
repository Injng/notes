1. Singular Value Decomposition

We would like to be able to write any matrix \(A\) as a product of simpler matrices (e.g. not just square invertible matrices with diagonalization). That is, for an \(m \times n\) matrix \(A\), we want to write it as:

\begin{align} \boxed{A = U \Sigma V^T} \end{align}

where \(U\) is an \(m \times m\) orthogonal matrix, \(\Sigma\) is an \(m \times n\) "diagonal" matrix, and \(V\) is an \(n \times n\) orthogonal matrix. This is known as the singular value decomposition (SVD) of a matrix.
