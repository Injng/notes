1. Error Handling

If an error occurs during load, the nearest +error.svelte file will be loaded. For example: 

<script lang="ts">
    import { page } from '$app/state';
</script>

<h1>{page.status}: {page.error.message}</h1>

2. Props

Use the $props rune to pass properties to components: 

<script lang="ts">
    let { adjective } = $props();
</script>

<p>this component is {adjective}</p>
